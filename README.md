# Премьера Стоматология

Одностраничный адаптивный сайт стоматологической клиники, разработанный на Next.js с TypeScript и SCSS Modules.

## Технологии

- **Next.js 16** - React фреймворк с App Router
- **TypeScript** - типизированный JavaScript
- **SCSS Modules** - модульные стили
- **React Hook Form** - управление формами
- **Zod** - валидация форм
- **Swiper** - слайдер/карусель для новостей
- **Vela Sans** - основной шрифт

## Структура проекта

```
app/
├── components/          # React компоненты
│   ├── Header/        # Шапка сайта с навигацией
│   ├── Values/        # Секция "Наши ценности"
│   ├── News/          # Секция новостей со слайдером
│   └── ContactForm/   # Форма обратной связи
├── api/               # API routes
│   └── contact/       # Обработка формы обратной связи
├── lib/               # Утилиты
│   └── api.ts         # Функции для работы с API
├── types/             # TypeScript типы
└── globals.css        # Глобальные стили и переменные
```

## Функциональность

- ✅ Адаптивная верстка для всех устройств
- ✅ Форма обратной связи с валидацией:
  - Обязательные поля: Имя, Телефон
  - Валидация формата телефона: +7 (423) 123-45-67
  - Валидация формата email
  - AJAX отправка формы
- ✅ Слайдер новостей с использованием Swiper
- ✅ SSR загрузка новостей из JSONPlaceholder API
- ✅ Адаптивное мобильное меню

## Запуск проекта

Установите зависимости:

```bash
npm install
```

Запустите dev-сервер:

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка для продакшена

```bash
npm run build
npm start
```

## Особенности реализации

- **SSR для новостей**: Данные загружаются на сервере для лучшего SEO
- **Валидация форм**: Используется Zod для схем валидации
- **SCSS Modules**: Изолированные стили для каждого компонента
- **TypeScript**: Полная типизация проекта
- **Адаптивность**: Мобильная версия с гамбургер-меню

## Переменные цветов

Основные цвета определены в `app/globals.css`:
- `--color-primary: #544B46` - основной цвет бренда
- `--color-accent: #8b4513` - акцентный цвет
